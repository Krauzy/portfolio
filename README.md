## Iniciar

```node
  node@v10.16.0 or higher
  npm@6.9.0 or higher
  git@2.17.1 or higher
```

## Como usar 

```sh
  # Clone esse repositório
  git clone https://github.com/Krauzy/portfolio.git

  # Vá para o respositório
  cd portifolio

  # Instale as dependências
  npm install

  # Inicie o servidor local
  npm start
```
